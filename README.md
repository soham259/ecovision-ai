# ♻️ EcoVision AI

**Smart Waste Classification and Recycling Assistant**

EcoVision AI is a Streamlit app that identifies waste items from a photo and answers natural-language questions about how to dispose of, recycle, reuse, or understand the environmental impact of that item — grounded in a curated recycling knowledge base.

---

## ✨ Features

- **📷 Image-based waste classification** — Upload a photo or take one with your camera; a fine-tuned MobileNetV2 model identifies the item's category with a confidence score.
- **🤖 AI Assistant** — Ask a photographed item questions in plain English. Responses are generated by **Gemini 2.5 Flash**, grounded in the item's knowledge base entry, so answers directly address whatever you actually ask instead of being limited to a fixed set of categories.
- **🧠 Intent detection** — A fine-tuned DistilBERT model tags each question with a topic (Disposal, Recycling, Reuse, Material, Environment, Decomposition), shown for reference and used as a fallback answer source if Gemini isn't configured.
- **📚 Knowledge base** — Structured facts per waste category: recyclability, recommended bin, material composition, environmental impact, decomposition time, tips, warnings, common reuse ideas, and more.
- **Fixed fallback mode** — If no Gemini API key is set, the app automatically falls back to the built-in knowledge-base lookup, so it always works out of the box.

## 🧩 Tech Stack

| Layer | Technology |
|---|---|
| Image classification | MobileNetV2 (fine-tuned, TensorFlow/Keras) |
| Intent detection | DistilBERT (fine-tuned, Hugging Face Transformers) |
| AI-generated answers | Gemini 2.5 Flash (`google-genai`) |
| Interface | Streamlit |

## 🚀 Setup

### 1. Clone and install dependencies

```bash
git clone https://github.com/your-username/voicevision-ai.git
cd voicevision-ai
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # macOS/Linux
pip install -r requirements.txt
```

### 2. Add your Gemini API key

Get a key from [Google AI Studio](https://aistudio.google.com/apikey), then set it up locally using **one** of the following (both are git-ignored):

**Option A — Streamlit secrets** (recommended for running the app):
```bash
mkdir .streamlit
```
Create `.streamlit/secrets.toml`:
```toml
GEMINI_API_KEY = "your-key-here"
```

**Option B — `.env` file:**
```bash
cp .env.example .env
```
Then edit `.env` and set:
```
GEMINI_API_KEY=your-key-here
```

> If no key is configured, the app still works — it automatically falls back to the built-in knowledge-base responses.

### 3. Run the app

```bash
streamlit run app.py
```

## 🖥️ Usage

1. Open **Image Prediction** or **AI Assistant** from the sidebar.
2. Upload a photo or take one with your camera.
3. Type your question (e.g. *"Can I recycle this?"*, *"How long does this take to decompose?"*).
4. View the detected item, recyclability, recommended bin, and a natural-language answer — plus material, environmental impact, and tips in the detail tabs.

Check **⚙️ System Status** in the sidebar to confirm whether Gemini AI answers are connected.

---

## 🔒 Security Notes

- Never commit `.env` or `.streamlit/secrets.toml` — both are excluded via `.gitignore`.
- If you're contributing, use `.env.example` as a template for required variables; don't fill it with real values before committing.
- If an API key is ever accidentally exposed (e.g. hardcoded and pushed), revoke it immediately in Google AI Studio and generate a new one.
